# Company Strategy Toolkit

## 1. North Star Metric (NSM)
ตัวเลขเดียวที่สะท้อนคุณค่าหลักให้ลูกค้า เช่น
- SaaS: MRR retained หรือ Weekly Active Teams
- บริการ: ลูกค้าที่ทำซ้ำใน 90 วัน
- ค้าปลีก: Basket value × ความถี่ซื้อต่อปี

ก่อนวาง strategy ต้อง agree NSM ก่อน — ทุกแผนกต้องโยงงานตัวเองให้ NSM ขยับได้

## 2. กรอบ OKR (Quarter)
```
Objective: <ข้อความปลายปิดสร้างแรงบันดาลใจ ทำ/ไม่ทำ ตอบได้>
Key Result 1: <metric + เป้า + กรอบเวลา>
Key Result 2: ...
Key Result 3: ... (เน้น 3-5 KR ห้ามเกิน)
```
เกณฑ์ KR ที่ดี: วัดได้, ท้าทาย 0.7 = ดีพอ, มี owner ชัด

## 3. SWOT แบบใช้ตัดสินใจ
| | Internal | External |
|---|---|---|
| ✅ | Strengths — อะไรที่เราชนะคู่แข่งจริง | Opportunities — เทรนด์/ช่องว่างที่กำลังเปิด |
| ⚠️ | Weaknesses — ที่ลูกค้าบ่นซ้ำ | Threats — ความเสี่ยงที่ทำให้รายได้หายเกิน 20% |

หลัง SWOT → ทำ matrix **SO/WO/ST/WT strategies** เพื่อแปลงเป็น action

## 4. Decision Memo (1 หน้า)
```
Context: 2-3 ประโยค สถานการณ์
Decision: เลือกอะไร
Why: 3 เหตุผลพร้อมข้อมูล
Risks: 2 ข้อ + mitigation
Owner / Deadline: ใคร / เมื่อไหร่
```

## 5. Scenario Planning
สร้าง 3 sceanrios: **Base / Upside / Downside**
- Base: สมมติฐานปัจจุบัน
- Upside: ดีลใหญ่ปิด + ตลาดบวก (+30% revenue)
- Downside: เสีย key account + ตลาดชะลอ (-25% revenue)
ดู cash runway ทั้ง 3 ทาง

## Checklist ก่อน delegate
- [ ] ระบุ desired output (เอกสาร / ตัวเลข / การตัดสินใจ)
- [ ] กำหนด deadline
- [ ] ระบุ "พอแค่ไหนคือพอ" (definition of done)
- [ ] บอกว่ารายงานกลับใครและรูปแบบไหน
