# Content Design Toolkit

## 1. Asset Brief Template
```
Subject: <สิ่งที่อยู่ในภาพ — ใครทำอะไรที่ไหน>
Style: <flat illustration / 3D / photo / minimalist / editorial>
Color palette: <hex 3-4 สี อิงแบรนด์>
Mood: <energetic / calm / premium / playful>
Composition: <rule-of-thirds / center / diagonal / overhead>
Lighting: <natural / studio / golden hour / dramatic>
Format: <aspect ratio + dimension>
Brand cue: <logo placement, font use, signature element>
Avoid: <ของที่ห้าม — เช่น stock people, hard sell, busy background>
```

## 3. Color Discipline
- ใช้ 1 primary + 1 accent + 2-3 neutrals — ห้ามเกิน 5 สีต่องาน
- คำนวณ contrast: text บนพื้น ≥ 4.5:1 (WCAG AA)
- Primary บนพื้นที่ ≤ 30% เพื่อให้สีไม่กลืน

## 4. Typography Pairing
- **Headline + Body** สองตัวพอ
- คู่ที่ใช้ได้: Inter (UI) + Playfair (heading) / Manrope + IBM Plex Serif / IBM Plex Sans Thai สำหรับ TH
- ขนาด headline:body อย่างน้อย 2.5x

## 5. Composition Checklist
- [ ] Subject อยู่ใน safe area (กันโดน UI ปิด)
- [ ] มี visual hierarchy — eye เริ่มจุดเดียวก่อน
- [ ] White space ≥ 25% ของพื้นที่
- [ ] Logo placement สม่ำเสมอ (มุมขวาบนหรือล่างเสมอ)
- [ ] CTA element เด่น (ปุ่ม/icon/arrow)
