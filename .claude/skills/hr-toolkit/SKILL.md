# HR Toolkit

## 1. JD Template (ที่ดี)
```markdown
# <ตำแหน่ง> — <ทีม> — <On-site / Hybrid / Remote>

## เราคือใคร (2-3 บรรทัด)

## คุณจะรับผิดชอบอะไร (5-7 bullet, action verb)
- ...

## ความสำเร็จ 90 วันแรก (วัดได้)
- 30 วัน: ...
- 60 วัน: ...
- 90 วัน: ...

## ที่จำเป็นต้องมี (must-have, 3-5 ข้อ)
- ...

## ที่จะดีมากถ้ามี (nice-to-have)
- ...

## ผลตอบแทน (ช่วงเงินเดือน + benefit เด่น 3 ข้อ)
```

## 2. Onboarding 30/60/90
**Day 0–7 (Setup)**
- [ ] อุปกรณ์/บัญชี/access พร้อม
- [ ] นัดเจอ buddy + ทีม
- [ ] อ่าน CLAUDE.md / handbook / OKR ปัจจุบัน
- [ ] เซ็น NDA + นโยบาย

**Day 30 (Learn)**: เข้าใจ product, process, ลูกค้า — เจอ stakeholder ครบทุกแผนก, ทำ shadow 3 ครั้ง

**Day 60 (Contribute)**: รับงานจริง 1-2 ชิ้นที่มี clear deliverable, ส่งของได้เอง

**Day 90 (Own)**: ถือ KPI หนึ่งตัว, นำ meeting 1 ครั้ง, มี improvement proposal 1 ข้อ

## 3. Performance Rubric (1-5)
- 5: เกินคาดอย่างต่อเนื่อง คนอื่นเอาเป็นแบบ
- 4: เกินคาดในบางด้าน
- 3: ทำตามคาดของตำแหน่ง
- 2: ต่ำกว่าคาดในบางด้าน → ต้องมี improvement plan
- 1: ต่ำกว่าคาดต่อเนื่อง → PIP หรือพิจารณาแยกทาง

วัด 4 มิติเท่ากัน: **Output / Quality / Collaboration / Growth**

## 4. นโยบายลา (สำหรับบริษัท SME ไทย)
- ลาพักร้อน: 10 วัน/ปี (เพิ่ม 1 วันต่อปีที่อยู่ครบปี max 15)
- ลาป่วย: 30 วัน/ปี (เกิน 3 วันใช้ใบรับรองแพทย์)
- ลากิจ: 3 วัน/ปี (ลาล่วงหน้า ≥ 1 วัน)
- WFH: 2 วัน/สัปดาห์ default ตกลงกับ manager

## 5. Exit Checklist
- [ ] รับ resignation letter
- [ ] กำหนดวัน last day + handover plan
- [ ] สัมภาษณ์ exit (เก็บ feedback)
- [ ] คืนอุปกรณ์ + revoke access วันสุดท้าย
- [ ] จ่ายเงินเดือนสุดท้าย + ใบรับรองทำงาน
