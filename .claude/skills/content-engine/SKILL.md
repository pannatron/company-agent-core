# Content Engine

## 1. Content Mix Rule (เลขทอง)
สมดุล 3 ประเภทใน calendar:
- **Educate 50%** — สอน, how-to, insight, ทำให้คนฉลาดขึ้น
- **Engage 30%** — ถามความเห็น, behind-the-scene, story
- **Promote 20%** — ขายตรง, โปร, launch

ถ้า promote > 30% ใน 4 สัปดาห์ติด → engagement จะร่วง

## 2. Hook Formula (3 วินาทีแรก)
- **Curiosity gap**: "เราเคยพังที่ X ครั้ง — สิ่งที่ได้เรียนรู้คือ..."
- **Contrarian**: "ทุกคนบอกให้ทำ A. เราทำตรงข้าม. ผลลัพธ์คือ..."
- **Stat shock**: "73% ของคนทำ X ผิดวิธี"
- **Specific promise**: "5 นาที. คุณจะรู้วิธี Y โดยไม่ต้อง Z"

## 3. SEO Checklist (per page)
- [ ] Primary keyword ใน title (≤ 60 ตัวอักษร) + H1
- [ ] Meta description 140-160 ตัวอักษร — ใส่ keyword + CTA
- [ ] URL slug สั้น มี keyword ไม่มี stopword
- [ ] H2/H3 มี LSI keyword (keyword ที่เกี่ยวข้อง)
- [ ] Internal link ≥ 2, external link ≥ 1 ที่น่าเชื่อถือ
- [ ] Alt text รูปทุกรูป
- [ ] First paragraph ใส่ primary keyword ภายใน 100 คำแรก
- [ ] Word count ≥ คู่แข่ง top 3 ใน SERP

## 4. Brand Tone (default)
- **Confident, not arrogant** — เรารู้ แต่ไม่อวด
- **Practical, not theoretical** — ทุกข้อต้องลงมือทำได้
- **Warm, not corporate** — ใช้คำ "เรา/คุณ" ห้าม "ทางบริษัท"
- **Concrete, not vague** — แทนคำว่า "เยอะ" ด้วยตัวเลข

## 5. เทมเพลตโพสต์ (LinkedIn / FB)
```
Hook (1 บรรทัด ดึงให้คนหยุด)

Context (3-5 บรรทัด เล่าเรื่อง)

3 takeaway:
1. ...
2. ...
3. ...

CTA (1 บรรทัด — comment / save / DM)
```

## 6. แคมเปญ — โครง 1 หน้า
- **เป้าหมาย** (วัดได้)
- **Audience** (เฉพาะเจาะจง 1-2 segment)
- **Key Message** (1 ประโยค)
- **Channel mix** + สัดส่วนงบ
- **Timeline** (kickoff → launch → wrap)
- **KPI** ที่จะวัด (อย่างน้อย 1 leading + 1 lagging)
