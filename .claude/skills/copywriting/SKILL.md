# Copywriting Toolkit

## 1. Hook Formulas (3 วินาทีแรก)
- **Curiosity gap**: "เราเคยทำ X 100 ครั้ง — สิ่งที่ได้เรียนรู้คือ..."
- **Contrarian**: "ทุกคนบอกให้ทำ A. เราทำตรงข้าม. ผลลัพธ์คือ..."
- **Stat shock**: "73% ของคนทำ X ผิดวิธี"
- **Specific promise**: "5 นาที. คุณจะรู้วิธี Y โดยไม่ต้อง Z"
- **Personal story**: "ปีที่แล้วเราเกือบเจ๊ง เพราะ..."
- **Question**: "ถ้าคุณมี <X>, คุณจะทำ Y มั้ย?"

## 2. Platform Character Rules
| Platform | First line | Total | Style |
|---|---|---|---|
| LinkedIn | 2 บรรทัด (≤ 200 ตัวอักษร) — hook + tease | 1,200-1,500 | Insight + story + lesson |
| Facebook | 80-120 — emoji 1 + CTA hint | ≤ 200 | Conversational, warm |
| Instagram | 125 ก่อน "see more" | 1,000-2,200 | Aspirational, hashtag 5-10 |
| X | hook ใน 60 chars | ≤ 280 | Punchy, 1 idea per tweet |
| TikTok caption | hook ใน 80 chars | ≤ 150 | Question + CTA "save this" |

## 3. Headline Patterns
- **How-to**: "วิธี <X> ใน <N> ขั้น"
- **List**: "<N> เหตุผลที่ <Y>"
- **Mistake**: "<N> ข้อผิดพลาดที่คน <persona> ชอบทำ"
- **Promise/Negation**: "ทำ <X> โดยไม่ต้อง <Y>"
- **Time-bound**: "ใน <timeframe> เราเรียนรู้ <insight>"
- **Identity**: "ถ้าคุณเป็น <persona>, <บอกอะไร>"

## 4. CTA Library
- **Awareness**: "Save it for later" / "Tag คนที่ต้องเห็น" / "ส่งให้เพื่อน"
- **Consideration**: "Comment 'X' เพื่อรับลิงก์" / "Subscribe newsletter" / "ดูตัวอย่าง"
- **Conversion**: "จองทดลองฟรี" / "DM 'เริ่ม'" / "สมัครก่อนใคร link in bio"

## 5. Brand Voice Checklist
- [ ] ใช้คำที่ลูกค้าใช้ ห้าม jargon ที่ไม่จำเป็น
- [ ] ใช้ "เรา/คุณ" ไม่ใช่ "ทางบริษัท / ผู้เรียน"
- [ ] มีตัวเลขเฉพาะ (X%, Y ครั้ง, Z วัน) อย่างน้อย 1
- [ ] ไม่อ้างคำคุณศัพท์ลอย ("ดีที่สุด", "ยอดเยี่ยม") โดยไม่มี proof
- [ ] ทุกย่อหน้ามี whitespace — ห้าม wall of text

## 6. โครง LinkedIn Post (มาตรฐาน)
```
Hook (2 บรรทัด — ตัด feed ได้)

Context (3-5 บรรทัด — เล่าสถานการณ์)

3 takeaways (bullet สั้นๆ):
→ ...
→ ...
→ ...

Lesson / Punchline (1 บรรทัด — บอก insight)

CTA (1 บรรทัด — บอกให้ทำอะไรต่อ)

#hashtag1 #hashtag2 #hashtag3
```
